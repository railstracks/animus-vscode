# Changelog

All notable changes to the Animus VSCode Extension will be documented in this file.

## [0.0.1] - 2026-07-21

### Added
- Initial release
- Connect to any Animus daemon (local or remote)
- Session list with timestamps, sorted by last activity
- Full chat interface with streaming responses (token-by-token)
- Thinking blocks rendered as collapsible message elements
- Tool calls rendered inline with results
- Attachment rendering (images, audio, video, text, files)
- Agent selection with automatic defaults loading (provider/model/reasoning)
- Per-message provider, model, and reasoning overrides
- Collapsible settings panel in chat view for mid-conversation changes
- New session form with agent/provider/model/reasoning controls
- Connection status indicator
- Auto-scroll during streaming with stop signal support
